## 使用说明

### 启动前端服务
```bash
cd frontend
npm install
npm run dev
```

### 访问新功能
1. **数据管理**: `/data` - 查看数据概览、管理文件、监控系统
2. **系统监控**: `/monitoring` - 专门的监控面板
3. **导航菜单**: 顶部导航栏新增"系统监控"选项

### 主要功能操作
1. **数据同步**: 在数据管理页面点击"同步数据"按钮
2. **进度跟踪**: 同步过程中自动显示进度模态框
3. **历史查询**: 点击"同步历史"查看过往记录
4. **系统监控**: 实时查看服务健康状态和性能指标
5. **文件管理**: 高级筛选、预览、批量操作数据文件
